# Relatch Landing Page Deployment

## Included
- index.html
- assets/Claude.svg
- assets/Codex.svg
- vercel.json

## Deployment Steps

1. Open your GitHub repo: relatchlp
2. Click "Upload files"
3. Drag and drop all files from this folder
4. Commit changes
5. Import repo into Vercel
6. Deploy

## Production URLs
- Landing Page: https://relatchlp.vercel.app/
- Frontend App: https://relatch-fe.vercel.app/
- Backend Proxy: https://claudly-proxy.vercel.app/
